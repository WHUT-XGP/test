# 二、起步

# 2.1 使用vue3

# 通过CDN形式使用Vue3

<script src="https://unpk.com/vue@next"></script>

# 通过@vue/cli使用vue项目

@vue/cli已经支持在创建项目的时候使用vue3了 在创建面板选择即可

# 全局安装@vue/cli,如果出错建议试试cnpm
npm install -g @vue/cli
vue create hello-vue3

创建的时候勾选vue3，其他操作同vue2的时候类似操作保持一致即可。

# 通过vite使用vue3

创建项目：

npm init vite-app hello-vue3
# OR
yarn create vite-app hello-vue3

安装依赖：

cd hello-vue3
yarn
# OR
cd hello-vue3
npm install

启动：

npm run dev
# OR
yarn dev
